# Personal Portfolio - Lanz Andrei C. Colico

Hi! I'm Lanz Andrei C. Colico. Welcome to my personal portfolio webpage, crafted with HTML, CSS, and JavaScript. Here, you'll find a showcase of my journey as a Computer Science student at TUP, featuring my projects, blogs, and more. Explore my work and get to know the skills and aspirations that drive my passion for technology.

## Features

- Modern, minimalist design with smooth animations
- Fully responsive layout for mobile, tablet, and desktop
- Interactive project gallery with hover effects
- Personal blog section
- Contact form with social media links
- Smooth scroll animations using Intersection Observer API

## Pages

- **Home** ([main.html](main.html)) - Landing page with introduction
- **Projects** ([projects.html](projects.html)) - Showcase of my work with images and descriptions
- **Blog** ([blog.html](blog.html)) - Personal stories and creative journeys

## Deployment to Vercel

This portfolio is optimized for deployment on Vercel. Follow these steps:

1. Push your code to a GitHub repository
2. Go to [Vercel](https://vercel.com) and sign in
3. Click "New Project" and import your GitHub repository
4. Vercel will automatically detect the configuration from `vercel.json`
5. Click "Deploy"

Your portfolio will be live at `your-project-name.vercel.app`

## Local Development

To run this project locally:

1. Clone the repository
2. Open `index.html` or `main.html` in your web browser
3. No build process required - it's pure HTML, CSS, and JavaScript
